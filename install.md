# 2. 온라인 레플 설치하기

온라인 래플은 CAFE24의 공식 디자인과 에디봇 디자인에서는 별도 설치 없이 사용하실 수 있습니다.\
공식 디자인을 사용하지 않으시는 경우 본 설치 매뉴얼의 방법을 통해 수동으로 설치해주셔야 합니다.

![공식지원 디자인 안내](<.gitbook/assets/image (3) (1).png>)

위 버튼을 클릭하시여 설치 메뉴얼을 다운로드 받으실 수 있습니다.
